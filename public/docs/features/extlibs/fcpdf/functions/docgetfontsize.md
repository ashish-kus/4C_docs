# DocGetFontSize()

## Purpose

FCPdf::`DocGetFontSize()` returns the default font size for the FCPdf document.

## Usage

```text
pointsize = FCPdf::DocGetFontSize(<docident>);
```

## Arguments

- `integer <docident>` — A valid FCPdf docident

## Returns

- `float <fontsize>` — The default font size for the document

## Where Used

FCPdf::`DocGetFontSize()` can be called anytime you have a valid FCPdf docident.

## Description

FCPdf::`DocGetFontSize()` returns the default font size for FCPdf document.
