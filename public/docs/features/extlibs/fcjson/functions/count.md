# Count()

## Purpose

FCJSon::`Count()` returns the total number of currently in use FCJSon objects for the 4csrvr process.

## Usage

```text
count = FCJSon::Count();
```

## Arguments

None

## Returns

- `integer <count>` — count of currently allocated FCJSon objects.

## Where Used

FCJSon::`Count()` can be called from anywhere.

## Example

```text
Example
```
