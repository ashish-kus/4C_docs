# ArrayAddArray()

## Purpose

FCJSon::`ArrayAddArray()` adds an array object to the end of another array object.

## Usage

```text
rc = FCJSon::ArrayAddArray(<arrayident0>,<arrayident1>);
```

## Arguments

- `integer <arrayident0>` — The array being added to
- `integer <arrayident1>` — The array being added

## Returns

integer `<rc>`
- `0` — OK
- `-1` — Error

## Where Used

FCJSon::`ArrayAddArray()` can be called anytime you have two different arrayidents.

## Example

```text
Example
```

## Description

FCJSon::`ArrayAddArray()` adds an array object to the end of another array object. The array object identified by `<arrayident0>` is extended by one.
