# DocGetFontFace()

## Purpose

FCPdf::`DocGetFontFace()` returns the default font facename for the FCPdf document.

## Usage

```text
fontface = FCPdf::DocGetFontFace(<docident>);
```

## Arguments

- `integer <docident>` — A valid FCPdf docident

## Returns

- `alpha <fontface>` — The default font facename for the document

## Where Used

FCPdf::`DocGetFontFace()` can be called anytime you have a valid FCPdf docident.

## Description

FCPdf::`DocGetFontFace()` returns the default font facename for FCPdf document.
