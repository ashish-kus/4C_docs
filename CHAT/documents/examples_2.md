- Example 1 - fcpdf.demo.1 This should be a simple program to follow. It allows you to select a font, a fontsize, x and y positions in points and some text to write. Clicking a button then will create one pdf file with one page and one line of text. Clicking another button will show the newly created PDF file in a client pdf viewer program.
- Example 2 - fcpdf.demo.2 This program shows a few more things that you can do, including setting stroke colors, drawing lines, and measuring text width. The program creates a pdf file of a red grid of a user chosen size. Row and column numbers or markers are printed in the outside grid boxes. You can actually create a form that works for measuring positions on another pre printed form.

All of the above examples are in the 4c demo application.
