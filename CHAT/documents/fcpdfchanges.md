## 05 September 2025

- There is a new funtion that makes it easier to split a long text into multiple lines and format it as a block of text within a specified rectangle. The function is [TextRect]({{< ref "Functions/textrect.md" >}})

## 28 July 2020

- The following 4 functions now have an optional final argument that allows specifying and encoding for the font being selected. This is most useful when using a UTF-8 font. [DocSetFontFace]({{< ref "Functions/docsetfontface.md" >}}) [DocSetFont]({{< ref "Functions/docsetfont.md" >}}) [PageSetFontFace]({{< ref "Functions/pagesetfontface.md" >}}) [PageSetFont]({{< ref "Functions/pagesetfont.md" >}})

## 05 February 2018

- True Type and Type 1 fonts are supported.
