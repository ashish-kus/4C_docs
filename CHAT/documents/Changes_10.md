## 22 December 2006

- When defining a trusted user in the AuthClient file, you can use "@Any" for a remote username to indicate any user on that machine.
- When defining a trusted user in the AuthClient file, you can use "@RemoteUser" for the local user name to indicate the local name is the same as the remote user name.
