## Purpose

FCPdf::`PageGetFontSize()` returns the current font size for the FCPdf document page.

## Usage

```text
pointsize = FCPdf::PageGetFontSize(<pageident>);
```

## Arguments

- `integer <pageident>` — A valid FCPdf pageident

## Returns

- `float <fontsize>` — The current font size of the page in points

## Where Used

FCPdf::`PageGetFontSize()` can be called anytime you have a valid FCPdf pageident.

## Description

FCPdf::`PageGetFontSize()` returns the current font size for the FCPdf document page.
