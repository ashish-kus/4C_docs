## Purpose

```text
xlview displays 4c shared memory
```

## Usage

```text
xlview <option> [ <optarg> ] ...
```

## Options

- `-c <comfile>` — This file is read in to shared mem
- `-d <dumpfile>` — Shared Mem is written to this file
- `-v` — verbose
- `-V` — Very Verbose
- `-H` — Help

## Returns

- `0` — Normal Exit
- `1` — Error Exit

## Example

```text
xlview -V
```
